# 🎯 Interview Judge System

An AI-powered interview evaluation platform designed to automate candidate assessment through resume analysis, technical evaluation, and intelligent scoring mechanisms.

---

## 🚀 Features

* 📄 Resume Upload & Analysis
* 🤖 AI-Based Candidate Evaluation
* 📊 Automated Scoring System
* 👨‍💼 Candidate Management
* 📝 Interview Performance Tracking
* 🔒 Secure Authentication & Authorization
* 📈 Performance Analytics Dashboard

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rajeshreddy021/InterviewJudgeSystem.git
```

### Navigate to Project

```bash
cd InterviewJudgeSystem
```

### Configure Database

Update application properties with your MySQL credentials.

### Run Application

```bash
mvn spring-boot:run
```
